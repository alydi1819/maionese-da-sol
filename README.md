# 🫙 Maionese da Sol

App de gestão de pedidos da Maionese da Sol, com impressão de etiquetas via Bluetooth (Niimbot XD 210X).

## ✨ Funcionalidades

- Cadastro e gestão de pedidos
- Histórico de pedidos concluídos
- Resumo de vendas com gráficos
- Impressão de etiquetas via Bluetooth (Niimbot)
- SMS automático para clientes
- Dados salvos no Supabase (nuvem)
- Funciona no celular e no computador

## 🚀 Como subir no GitHub Pages

### 1. Criar o repositório

1. Acesse [github.com](https://github.com) e faça login
2. Clique em **"New repository"** (botão verde)
3. Nome: `maionese-da-sol`
4. Deixe como **Public**
5. Clique em **"Create repository"**

### 2. Fazer upload do arquivo

1. Na página do repositório recém-criado, clique em **"uploading an existing file"**
2. Arraste o arquivo `index.html` para a área de upload
3. Clique em **"Commit changes"**

### 3. Ativar o GitHub Pages

1. Vá em **Settings** (aba no topo do repositório)
2. No menu lateral, clique em **"Pages"**
3. Em **"Branch"**, selecione `main` e a pasta `/ (root)`
4. Clique em **"Save"**
5. Aguarde ~1 minuto

### 4. Acessar o app

Seu app estará disponível em:
```
https://SEU_USUARIO.github.io/maionese-da-sol
```

## 📱 Como usar no celular

1. Abra o link acima no **Chrome** do celular
2. Toque no menu ⋮ → **"Adicionar à tela inicial"**
3. O app vai aparecer como ícone no celular, igual um app normal!

> ⚠️ Use sempre o **Chrome** para que o Bluetooth funcione.

## 🖨️ Impressora Bluetooth

- Modelo: Niimbot XD 210X
- Toque no ícone Bluetooth no topo do app
- Selecione a impressora na lista
- O app funciona em `https://` então o Bluetooth é liberado pelo Chrome

## 🗄️ Banco de dados

Os dados são salvos no [Supabase](https://supabase.com) automaticamente.
